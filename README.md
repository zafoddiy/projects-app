# Projects Webapp
This project is based on a Telia internship homework assignment. In addition to making the html work, I've added additional functionality to see the database in a readable manner, as well as styling.
This repo provides files for both the server-side (Flask) and client-side (React.tsx)
# Requirements
Server-side (Flask) requires sqlite3, flask, flask-CORS. Client-side (React) requires vite, react, react-dom-router.
# Usage
Run the server-side app.py file first. Use vite to run the client-side app.
